New-HostsFileEntry

SYNOPSIS

Adds an entry to the HOSTS file for the given ip address and hostname

SYNTAX

New-HostsFileEntry [-IPAddress] <String> [-Hostname] <String> [[-Note] <String>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
